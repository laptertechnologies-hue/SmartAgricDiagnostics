# SmartAgricDiagnostics
Since we are using the intelligent Demo Mode while you train the real model via the Python pipeline, you can still test the entire flow:

Tap on Maize, Cassava, or Beans.
Point your camera at anything (a leaf, your keyboard, etc.) and tap Diagnose.
See the simulated diagnosis, confidence score, and specific treatment recommendations.
Tap Save to History.
Check the Diagnosis History screen to see your saved records.
Let me know how it looks on your device or if you'd like to work on the Python ML pipeline next to train the real model!
